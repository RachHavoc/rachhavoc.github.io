Machine Overview

Difficulty: Intermediate

Platform: Offensive Security Proving Grounds

Description: The “Cockpit” VM is an intermediate-level Linux machine created by OffSec for penetration testing training. It tests reconnaissance, web exploitation and privilege escalation skills, culminating in a tar wildcard vulnerability exploit to gain root.

Key Features

  • Reconnaissance of multiple services (HTTP, HTTPS, SSH).
  • SQL Injection to bypass a web login and harvest credentials.
  • Privilege escalation via tar wildcard exploit.

Initial Reconnaissance

sudo nmap -sC -sV 192.168.236.10 -oN nmap-cockpit

All ports:

sudo nmap -p- -sC -sV 192.168.236.10 -oN nmap-cockpit

Enum /js/ directory:

Re-run with likely file extensions:

feroxbuster -u http://192.168.236.10/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,txt -o ferox.rerun

Generated SSH key and pasted public key:

ssh james@192.168.236.10 -i id_james

Privilege Escalation

sudo -l

User can run tar as root. Exploit tar wildcard vulnerability:

Helpful guide 
sudo /usr/bin/tar -czvf /tmp/backup.tar.gz *

Key Takeaways

  • Always re-run directory scans with likely file extensions.
  • SQL Injection can still bypass login even with filters if syntax is adjusted.
  • Tar wildcard privilege escalation is a classic and powerful technique.
