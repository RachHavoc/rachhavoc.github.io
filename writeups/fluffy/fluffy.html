Release Arena

Description:
Fluffy is an easy Windows machine designed around an assumed breach scenario. Initial credentials are provided for a low-privileged user. Exploiting CVE-2025-24071 allows obtaining credentials for another low-privileged user. Enumeration uncovers ACLs on the winrm_svc and ca_svc accounts. Using WinRM with winrm_svc and exploiting the Active Directory Certificate Service (ESC16) with ca_svc ultimately leads to Administrator access.

Remediation Recommendations

  • Enforce strong, unique passwords for all service accounts.
  • Limit LDAP/WinRM write permissions to strictly necessary users.
  • Monitor CA certificate requests and log anomalous enrollments.
  • Implement Kerberos ticket auditing to detect abnormal TGT requests.
  • Regularly rotate service account credentials and revoke unused accounts.
  • Apply principle of least privilege to administrative groups.
