Machine Overview

Difficulty: Intermediate

Platform: Offensive Security Proving Grounds

Description: The Extplorer virtual machine (VM) is a retired Linux box designed to teach web application penetration testing and privilege escalation skills. The box features a WordPress installation alongside the eXtplorer file manager, which provides opportunities for credential exploitation, file upload vulnerabilities, and local privilege escalation.

Key Learning Objectives

  • Web server enumeration and directory discovery
  • Credential harvesting and cracking
  • File upload exploitation and reverse shell creation
  • Privilege escalation through group permissions and debugfs

Enumeration

Nmap Scan

Initial scan:

sudo nmap -sC -sV 192.168.236.16 -oN nmap-extplorer

All ports scan:

sudo nmap -p- -sC -sV 192.168.236.16 -oN nmap-extplorer-all-ports

WordPress detected on port 80.

Web Enumeration

Homepage revealed:

Feroxbuster

feroxbuster -u "http://192.168.236.16" -w /usr/share/seclists/Discovery/Web-Content/common.txt --depth 2 --filter-status 404

/filemanager directory discovered, leads to login page.

Credentials & User Dora

Dora’s password hash found in configuration file:

cat /var/www/html/filemanager/config/.htusers.php

Cracked hash with hashcat:

hashcat -m 3200 dora.hash /usr/share/wordlists/rockyou.txt

Password found: doraemon

Privilege Escalation

Dora belongs to disk group, which allows root-level access through debugfs:

Box Summary

The Extplorer VM teaches a complete workflow for web application exploitation and Linux privilege escalation. Learners practice enumeration, credential cracking, file upload exploits, and root escalation via misconfigured permissions.

  • Initial Access: Discover eXtplorer through web enumeration, login using credentials or default passwords.
  • Web Exploitation: Upload PHP webshell to gain a reverse shell.
  • User Privileges: Identify hashes for application users and crack them for access.
  • Privilege Escalation: Exploit membership in disk group and debugfs to extract root hash, then crack to become root.
